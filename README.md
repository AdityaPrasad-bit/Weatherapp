# Weatherapp

In this app I have search the particular country and then it will fetch the weather of that particular country. 

With the help of data Task(Creates a task that retrieves the contents of the specified URL, then calls a handler upon completion) it will perform there operation:-

1)Data.
2)Response.
3)Error.

Declaration;-struct URL
URL:-A value that identifies the location of a resource, such as an item on a remote server or the path to a local file.

Declaration:-class URLSession : NSObject
URLSession:-An object that coordinates a group of related, network data-transfer tasks.

Declaration:-class JSONDecoder
JSONDeoder():-An object that decodes instances of a data type from JSON objects.
